운영체제 - 컴퓨터 하드웨어 시스템을 운영하는 소프트웨어

스마트폰 - 컴퓨터라고 할 수 있다.
계산기 - 컴퓨터라고 할 수는 없다.
컴퓨터 - 정보를 처리하는 기계

정보?
I(x) = -logP(x) - Claude Shannon
사건 x의 정보량 - I(x)
사건이 발생할 확률 P(x)

주사위를 던지는 상황..
주사위를 던져서 3이 나왔다 -> 정보
A라는 사람이 그 정보를 B에게 전달하는데, 그 정보의 양은?
B가 A가 알고 있는 정보를 맞힐 확률은 1/6
log(6)의 정보...

동전을 던지는 상황
정보의 양은 1
정보의 단위는? 1 bit(binary digit)

불확실한 것을 측정해서 표현한 것

컴퓨터가 정보를 처리하는 방법
8bit -> 1byte ...
정보를 처리하기 위해서는 정보의 상태를 변환하는 장치가 필요하다. Boolean Algebra - NOT AND OR
게이트를 만들면 된다.(트랜지스터)

정보의 저장: 플립-플롭
데이터 전송: 데이터 버스

덧셈 - 반가산기, 전가산기(AND, OR 게이트..)
뺄셈 - 2의 보수 표현법
곱셈/나눗셈 - 덧셈 뺄셈의 반복
실수 연산 - 부동 소수점 표현법
함수 - GOTO

컴퓨터 - 범용성이 있다.
NOT AND OR 게이트 만으로 모든 계산을 할 수 있다.
NAND 게이트만으로도 모든 계산을 할 수 있다.
general-purpose computer - 이런 것을 컴퓨터라고 부른다.

계산가능성 computability
Turing-computable: 튜링머신으로 계산 가능한 것
정지 문제 Halting Problem: 튜링머신으로 풀 수 없는 문제 중 하나

튜링 머신         <->   현대 컴퓨터
튜링머신          <->   응용프로그램
유니버셜 튜링머신  <->   운영체제
헤드              <->   CPU
테이프            <->   메모리

stored-program(내장형 프로그램 방식) - 폰 노이만
메모리에 프로그램을 저장하는 컴퓨터

소프트웨어를 RAM에 저장한다.
프로그램에 있는 명령어를 CPU가 fetch - execute
stored-program : fetch - execute 사이클을 갖고 실행
ISA(instruction Set Architecture) - 명령어 집합 아키텍쳐

프로그램 - Set of Instruction, 명령어의 집합 (컴퓨터 하드웨어에게 특정 task를 실행시키는 명령어의 집합)
컴파일러가 소스코드를 컴파일 -> 기계어
기계어도 0과 1, 정보로 표현된다.
기계어를 어셈블리어로..(기계어를 읽을 수 있도록)

운영체제도 프로그램
- 컴퓨터에서 항상 실행되는 유니버셜 튜링머신
- 시스템 서비스를 애플리케이션 프로그램을 제공한다.
User <-> App <-> OS <-> HW

운영체제가 해야할 일
- 프로세스, 리소스, 유저 인터페이스(마우스 등) 등을 관리한다.

운영체제란? - 컴퓨터 시스템을 운영하는 소프트웨어

아두이노, 라즈베리파이 - 하드웨어를 프로그래머가 직접 다룬다.
