mkdir: 디렉터리 생성
mkdir -p: 중첩된 디렉터리 한 번에 생성

리눅스에서는 디렉터리와 파일을 동등한 존재로 다룬다.
파일과 디렉터리의 이름이 중복될 수 없다.

touch: 파일 생성

touch는 파일의 타임스탬프를 갱신하는 명령어.
파일이 존재하지 않으면 새롭게 파일을 만든다.
이미 존재하는 파일을 지정해도, 내용이 지워지거나 바뀌지 않는다.

rm: 파일삭제
rm -r: 재귀적으로 디렉터리 트리를 삭제한다.
rm -i: 삭제할 지 물어보는 옵션
rmdir: 빈 디렉터리를 삭제한다. 숨겨진 파일이 있어도 삭제되지 않는다.

cat: 파일 내용 출력
cat -n: 행 번호 출력

less: 위아래로 스크롤 하며 파일 볼 수 있게 출력
f: 한 화면 아래로 스크롤
b: 한 화면 위로 스크롤
j: 한 행 아래로 스크롤
k: 한 행 위로 스크롤
q: less 종료
/: 문자열 검색(아래)
?: 문자열 검색(위)
n: 다음 검색결과
N: 이전 검색 결과

cp: 파일 복사
cp -i: 덮어쓰기 전에 확인
cp -r: 디렉터리를 복사할 때 재귀적으로

mv: 파일 이동
-r 옵션 없이도 재귀적으로 내부 디렉터리 모두 이동

ln: 하드링크를 생성한다. 한 원본 파일에 여러 개의 이름을 붙이는 것이다. 모든 하드링크가 없어져야 파일이 삭제된다.
ln -s: 심볼릭링크를 생성한다. 원본 파일을 삭제하면 링크가 깨진다.
